# 产品规格

## 产品定位

口袋记账 AI 版是一个面向学生和年轻用户的轻量个人财务工具。它不追求企业级财务管理，而是解决“快速记账、看懂本月钱花到哪里、预算快超时有人提醒”这三个问题。交互上采用分页面工作台，而不是把所有模块放进一个长页面里下滑查看。

## 竞品借鉴

- YNAB：预算意识强，强调钱应该有去向。
- Monarch Money：总览、交易、报表组织清晰。
- Copilot Money：自动分类和消费洞察体验强。
- 国内记账应用：快速记账、分类统计、多账户和月度报表是用户熟悉的基本能力。

## 第一版模块

- Dashboard：本月收入、支出、预算剩余、AI 建议、趋势图。
- Quick Entry：一句话对应一笔账，AI 解析后由用户确认；收入和支出分别入账，净额由统计计算。
- Transactions：流水列表，按日期分组，支持编辑、删除和自定义标签。
- Budget：设置月度预算，显示风险状态。
- Analytics：分类占比、周/月趋势、账户分布，并提供文字结论。
- Settings：真实保存主接口、备用接口、API Key 和超时时间；可测试主备模型连接；公开状态只显示是否配置，不回显真实 Key。

## 页面结构

- 桌面端：左侧深色导航固定展示模块入口，右侧只渲染当前页面。
- 移动端：导航收缩为网格式入口，核心页面仍保持独立切换。
- 设置作为独立页面，不塞在总览或底部折叠区。

## 不做什么

- 不做登录和多用户。
- 不做银行账单自动同步。
- 不做扫码、发票 OCR 和支付平台导入。
- 不做原生 App。
- 不做复杂资产负债表。
- 第一版不做单句批量多笔解析，也不在解析阶段把收入和支出抵消成一笔账。

## 核心用户流程

1. 用户打开页面，看到本月消费和预算状态。
2. 用户输入一句自然语言记账文本。
3. 系统解析为结构化账单。
4. 用户确认或修改字段。
5. 系统写入 SQLite。
6. Dashboard 和图表实时更新。
7. 用户通过标签和日期分组复盘具体消费场景。
