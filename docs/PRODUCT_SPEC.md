# 产品规格

## 产品定位

口袋记账 AI 版是一个面向学生和年轻用户的轻量个人财务工具。它不追求企业级财务管理，而是解决“快速记账、看懂本月钱花到哪里、预算快超时有人提醒”这三个问题。

## 竞品借鉴

- YNAB：预算意识强，强调钱应该有去向。
- Monarch Money：总览、交易、报表组织清晰。
- Copilot Money：自动分类和消费洞察体验强。
- 国内记账应用：快速记账、分类统计、多账户和月度报表是用户熟悉的基本能力。

## 第一版模块

- Dashboard：本月收入、支出、预算剩余、AI 建议、趋势图。
- Quick Entry：一句话记账，AI 解析后用户确认。
- Transactions：流水列表，支持筛选、编辑、删除。
- Budget：设置月度预算，显示风险状态。
- Analytics：分类占比、周/月趋势、账户分布。

## 不做什么

- 不做登录和多用户。
- 不做银行账单自动同步。
- 不做扫码、发票 OCR 和支付平台导入。
- 不做原生 App。
- 不做复杂资产负债表。

## 核心用户流程

1. 用户打开页面，看到本月消费和预算状态。
2. 用户输入一句自然语言记账文本。
3. 系统解析为结构化账单。
4. 用户确认或修改字段。
5. 系统写入 SQLite。
6. Dashboard 和图表实时更新。
